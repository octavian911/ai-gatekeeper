# Security Policy

## Reporting a Vulnerability
If you believe you have found a security vulnerability, please report it responsibly.

- Email: security@ai-gatekeeper.ca
- Please include: affected URL, steps to reproduce, impact, and any proof-of-concept details.
- Do not publicly disclose until we have confirmed and mitigated the issue.

## Scope
- https://ai-gatekeeper.ca (business site)
- https://app.ai-gatekeeper.ca (web app)
- Any APIs used by the app

## Response Targets
We aim to acknowledge reports within 72 hours and provide a remediation plan as soon as possible.

Thank you for helping keep users safe.
